Formulario en HTML

El objetivo de los formulrios en HTML es crear la comunicación o interactividad con el usuario final

  • input: se usa para crear controles interactivos con la finalidad de recibir datos del usuariosa

      • Atributos

    1. type: tipo de información que quieres recibir sirve para pder interpretarla como tal. Por ejemplo Text, number, password, email, etc.
    2. Placeholder: podemos guiar al usuario escribiendo la instrucción o la información que tiene que poner.
    3. maxlength: largo máximo de caracteres que va admitir el input.
    4. required: hace obligatorio el input al que se lo asignes volviendo al input faltante. Validación sencilla.
    5. name: sirve para identificar los inputs en archivos como php
    6. id: sirve para darle un valor como identificadro unico.
    7. for: me da pie a que empiecen a escribir en el input selecionado
